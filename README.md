+ __Cardinal Spline Curve and Motion Control__

Windows / Qt

用户可以选定点，采用Cardinal样条曲线算法进行曲线插值，并加入了弧长参数化处理以实现插值点的均匀分布。然后可以在路径上防止小车使其按照路径进行仿真运动。
